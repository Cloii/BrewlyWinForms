# 🧠 BrewlyPOS: A Comprehensive Point of Sale System
BrewlyPOS is a robust and feature-rich point of sale system designed to streamline retail operations, manage inventory, and process transactions efficiently. This project aims to provide a user-friendly and intuitive interface for managing products, orders, and payments, making it an ideal solution for businesses of all sizes.

## 🚀 Features
* **Product Management**: Easily manage products, including adding, editing, and deleting items, as well as tracking inventory levels and product details.
* **Order Management**: Manage orders, including creating, editing, and deleting orders, as well as tracking order status and history.
* **Payment Processing**: Process payments securely and efficiently, with support for various payment methods.
* **Inventory Management**: Manage inventory levels, track product quantities, and receive low-stock alerts.
* **User Interface**: Intuitive and user-friendly interface, making it easy for employees to navigate and use the system.

## 🛠️ Tech Stack
* **Frontend**: Windows Forms, .NET Framework
* **Backend**: .NET Framework, C#
* **Database**: JSON files for data storage
* **Libraries**: Newtonsoft.Json for JSON serialization and deserialization
* **Dependencies**: System, System.Collections.Generic, System.Drawing, System.IO, System.Net, System.Windows.Forms
